RGBA Color
I used RGBA color in the div section for the description lists in my web page. This allowed me to easily make the background color partially transparent, while keeping the text dark.

CSS3 Opacity Property
I used the CSS3 Opacity Property to make the H1 and H2 sections of my page (the visual title area) partially transparent. This made both the background and font color partially transparent.

Background-Image
I used the background-image property to set the background for each of my pages, and used the background-size proptery to adjust the scaling.
